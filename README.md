# stecknologies.com

The official homepage for Stecknologies, Inc. [View here](https://stecknologies.com)

To run locally:
1) Install dependencies
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

2) Run locally
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
    ```

3) Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
